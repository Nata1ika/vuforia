Задача:
 
Реализовать приложение для андроид на платформе Unity3D с реализацией
дополненной реальности. Библиотека для реализации дополненной
реальности Vuforia.

Основной функционал:
- определять минимум 1 маркер.
- на маркере должна выводиться любая 3Д модель.
- при потере маркера модель и все с нею связанное должно отключатся.

Требования к 3Д модели:
- должна иметь минимально достаточный набор анимаций для реализации ее
поведения. (модель и анимации можно скачать из стора юнити, там много
бесплатных моделей)
- анимирование модели осуществить с помощью аниматора (создать свой,
минимально необходимый функционал)
- добавить взаимодействие с моделью(тапы, джойстик.) для придания
интерактивности.
- минимальная озвучка сцены
- добавить системы частиц.
Все это должно без корректировок собираться на андроид.
